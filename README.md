# KintoneOpenToday

## Kintone 本日日付(半)自動更新プラグイン

## 1.概要

指定した日付を開いた(開いている)時に更新するプラグインです。

更新タイミングは、開いた時と表示で一日経過した時です。

更新までのタイマーは非表示にも出来ます。

## 2.注意点

上位の説明文に更新した日付を保存しますので、一意の文字を設定してください。

保存した日付の表示は直ぐに反映しない場合があります。
